## What was built (summary, read the spec for editorial direction)

- **Eleventy 3.x** static site, deploys via GitHub Actions (`.github/workflows/deploy.yml`) on push to `main`.
- **Hybrid landing** at `/`: hero (with self-hosted video gate), pitch + 4 features, "Transmissions" screenshots row, Latest Transmissions devlog cards, "Watch the trailer" YouTube embed (gated), Stay-in-the-loop callout (Buttondown form gated, Discord block always renders), footer.
- **Sticky bottom CTA** on every page: coequal Steam (cyan) + Discord (pink) buttons. Discord copy is "Help shape the game" — editorial framing, do NOT change to "Join Discord" or similar.
- **Devlog feed** at `/devlog/`: post grid with in-page type filter (All / Origins / Tools / Devlog).
- **Post pages** at `/devlog/<slug>/`: type badge, title, date meta with optional "originally built [month]" honesty note, optional cover image, body, footer with Buttondown signup + back link.
- **Three post types:** `origins` (looking-back, 4-6 total), `tools` (one per dev tool, 15-25+), `devlog` (ongoing). Each has its own colored badge.
- **Draft gating:** `published: true` flag in frontmatter. In production builds, posts without it are excluded; in `npm run dev` (Eleventy `--serve` mode), all posts visible. There's also a local-only `/drafts/` index page.
- **Atom feed** at `/feed.xml`.
- **Cloudflare Web Analytics** beacon (gated on token in site data).

## Important deviations from the original plan (controller-accepted)

These differ from what the plan template said because the plan template had bugs the implementer correctly worked around. Don't try to "fix" them back to the plan's text:

1. **Eleventy 3.x env-var behavior.** Eleventy 3.x sets `process.env.ELEVENTY_RUN_MODE` itself based on the CLI subcommand (`build` / `serve` / `watch`), overwriting any shell-supplied value. The draft-gating logic in `src/posts/posts.11tydata.js` and `src/drafts.11tydata.js` accepts dev/serve/watch modes AND an `ELEVENTY_INCLUDE_DRAFTS=true` opt-in for non-interactive verification (e.g. `ELEVENTY_INCLUDE_DRAFTS=true npm run build` builds with drafts visible).

2. **Permalink date prefix.** `fileSlug` strips date prefixes in 3.x. The post permalink is computed in `posts.11tydata.js` to preserve the full filename basename (so `2026-05-15-intro.md` → `/devlog/2026-05-15-intro/`).

3. **RSS plugin v2 pin.** `@11ty/eleventy-plugin-rss@v3` is ESM-only, project is CJS. Pinned to `^2`.

4. **Drafts page sidecar.** Eleventy 3.x doesn't honor a Nunjucks-rendered `"false"` string as the boolean-skip permalink. `src/drafts.11tydata.js` is a JS sidecar that gates the permalink with a real boolean.

5. **Unused `isDevMode` global** in `.eleventy.js`. Registered but ends up unused (sidecar implements its own check). Left in place.

6. **`script.js` deleted.** Original had an IntersectionObserver pausing the hero video offscreen — minor optimization, dropped per plan instruction.

## Known weirdness

- **CRLF warnings** on every git operation. Windows line endings on a repo that doesn't have a `.gitattributes` policy. Cosmetic, not a problem.
- **Stale node processes** can pile up if `npm run dev` isn't Ctrl+C'd cleanly (Tim hit this overnight — port 8080 still bound this morning from yesterday's session). Diagnostic: `netstat -ano | grep :8080`. Cleanup: `taskkill //F //PID <pid>`.
- **The `<a>` wraps `<article>`** in `post-card.njk`. HTML5 allows interactive content wrapping flow content here, but some linters flag it. Not a bug.
